Exercise

The following tasks help you to build your first prototype.

  • Query movie data from the given SQLite database (Release date, Running time, Budget, Box office).
  • Save it in the file system as CSV using Pandas' Dataframe.
  • Build a UI module that is able to show the movie data.
  • Add a button to load the movie data into the UI.
  • Allow to filter the movie list in the UI.
  • Add a 2x2 matrix with axes Budget and Box Office and the number of movies that belong to a cell.
  • Color the cells according to the percentage of the share the cell's movies have of all movies.
  • Add two actions to the movie list: Delete movie from the list, Show movie details.
  • Visualize the formula you have used to calculate the percentages in the matrix.
  • Separate movie list, movie details, movie statistics in three modules.
