AngularJS

There are numerous AngularJS features and components but you need to know only a handful of them.

Attributes

With the following attributes one can cover 99% of the requirements of a prototype.

  • ng-model - binds the input element to a variable
  • ng-if - defines the condition under which the HTML element is shown
  • ng-repeat - iterates through an array
  • ng-class - defines conditions under which CSS classes at the element shall be added
  • ng-click - defines the method to be called when the HTML element is clicked
  • ng-init - initializes a variable name, useful in nested loops to name the index variable
  • ng-include - includes an HTML fragment in the page

The following attributes are mandatory to set up AngularJS in the page, for a prototype you need them only once in the page.

  • ng-app - marks the root element of the HTML markup AngularJS takes care of
  • ng-controller - once per module
