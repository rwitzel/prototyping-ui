Good Practices

  • Use milliseconds since epoch instead of JavaScript Datw objects in objects that have to be stored in local storage.
  • Cache results in the backend.
  • Allow the user to reset all inputs with defaults.
  • Make sure the user can hide undesired information.
  • Add a legend to explain abbreviations and numbers. Or use mouseovers.
  • Provide links to more information. Don't duplicate information that can be found somewhere else.
