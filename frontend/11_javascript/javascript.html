JavaScript

Language Version

Go with ES6, i.e. use const, let, () => expression.

let var1 = 456;
const immutable1 = (x) => x*x;

APIs

There are numerous JavaScript APIs but you don't need hardly any of them.

Useful APIs:

  • making HTTP calls: fetch or AngularJS' $http
  • working with JSON: JSON.stringify and JSON.parse
  • asynchronous calls: Promise
  • saving data: localStorage.getItem and localStorage.setItem
  • logging: console.log
