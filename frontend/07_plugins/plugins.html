Plugins

Native Plugins

Just include the downloaded plugin (usually just a JavaScript file) in the index.html

Then register the directive in the application.

Example: The plugin AngularJS Dropdown Multiselect is using Bootstrap.

Wrappers

Example: The jQuery plugin Charts.js can be used in AngularJS via wrapper.

jQuery

You must create a directive to link jQuery with AngularJS.

Example: The jQuery plugin jquery.toc creates from the headline markup in the page a table of contents. Note that the headlines of the subsequent sections are missing.
