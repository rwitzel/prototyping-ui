CSS

Basic rule: Don't try to style your page. CSS is too complicated for a novice. Instead ask your colleage to style your page. Or even better, use only Bootstrap classes.

Selectors

<div class="module1"> <table class="review"> <tr> <td> <span class="currency_symbol">Text</span> </td> </tr> </table> </div>

There are numerous ways to select a span element with class currency_symbol in a table cell in Module 1. 

div.module1 table.review td   span.currency_symbol             { color: red;   }
   .module1 table             span.currency_symbol             { color: green; }
                                  .currency_symbol             { color: blue;  }
   .module1              td > span.currency_symbol             { color: black; }
   .module1                   span.currency_symbol, .module1 p { color: grey;  }

Properties

There are 370 CSS properties but you need only a few of them for a prototype.

  • color: red; Text color
  • border: 1px solid black; Border
  • padding-left: 5px; Inner space
  • margin-bottom: 0px; Outer space
  • width: 400px; Width
  • font-weight: bold; Bold
  • white-space: nowrap; No line break
  • background-color: lightblue; Background color
  • font-size: xx-small; Smaller font size
  • display: inline-block; Blocks within the text flow (something between div and span)
  • float: left; Separate text flow
  • text-align: center; centered text, e.g. in table cells
  • border-collapse: collapse; for tables the borders between table cells are collapsed

!Important

Styles from third-parties like Bootstrap can be overridden: any-property-name: any-property-value !important;

Gradients

One can easily create dynamic gradients with background-color: hsl({{hue}}, 50%, 50%);.
